Privacy Policy

Last updated: 2026-04-18

video-test is a personal, single-user automation tool. It does not operate as a public service and does not collect data from third parties.

1. Data we process

When the operator (the only user) signs in with TikTok via the Login Kit, video-test receives and stores the following information locally on the operator's machine:

  • OAuth access and refresh tokens
  • The operator's TikTok open_id, union_id, display name and avatar URL

This information is kept in a local PostgreSQL database on the operator's own server and is never transmitted to any third party.

2. Content

video-test downloads publicly accessible video URLs supplied by the operator, processes them locally, and uploads the result to the operator's own TikTok account via the Content Posting API. No content belonging to other TikTok users is accessed or stored.

3. Third-party services

Outbound requests are made to: TikTok (Login Kit / Content Posting API) and OpenRouter.ai (LLM text generation). No user data beyond the OAuth flow described above is sent to these services.

4. Retention

Tokens and metadata are retained until the operator deletes the local database or revokes access from within TikTok's app settings.

5. Rights

The operator can revoke the application's TikTok access at any time via TikTok's privacy settings. Doing so will invalidate the stored tokens.
